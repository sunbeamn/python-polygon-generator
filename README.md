# Polygon Drawer with Error Handling

This is a Python project I created while finishing 9th grade. I am learning Python and this is my first step into building stable programs.

## Credits & Logic
- **Core Algorithm:** I used the standard geometric logic (`360 / n`) for drawing polygons. I took this specific drawing function from my online course -which is took from internet resources.-
- **My Contribution:** I took that "raw" code and added a **Robust Error Handling** layer. I implemented `try-except` blocks to ensure the program doesn't crash when a user enters invalid data.

## Features
- Draws any polygon based on user input.
- Prevents crashes using `try-except` (my main focus in this project).
- Uses the `turtle` library for visualization.
